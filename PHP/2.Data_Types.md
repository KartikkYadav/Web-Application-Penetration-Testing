# Data Types

a. Integer
   
   represents whole numbers (positive or negative).
   
   $age = 25;


b. Float (Double)
   
   Represents decimal or floating point numbers.
   
   $price = 99.99;


c. String
   
   A sequence of characters, enclosed in single or double quotes.
   
   $name = "John Doe";


d. Boolean
  
   Represents either true or false.
   
   $isAdmin = true;
